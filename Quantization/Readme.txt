*********************************************

Quantization, involved in image processing, is a lossy compression technique achieved by compressing a range of values to a single quantum value.

*********************************************

=> To compile the code run the following line in command prompt:

	javac A2.java



=> To run code

	java A2 image1.raw 16

		image1.raw : image file name
		16 : The number of colours in quantized image(in powers of 2)
			(can be-> 2, 4, 16, 32, 64...)

All the 2 arguments after the java class name must be provided.
