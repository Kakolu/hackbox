Readme

*********************************************

=> To compile the code run the following line in command prompt:

	javac image_reader.java



=> To run code

	java image_reader Image1.rgb 1 1 1 256

		Image1.rgb : RGB image file name
		1 : Extent of Y sampling
		1 : Extent of U sampling
		1 : Extent of V sampling
		256 : Quantization value


All the 5 arguments after the java class name must be provided.
